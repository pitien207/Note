# Note

A small, elegant note-taking website built with plain HTML, CSS, and JavaScript.

## Structure

- `index.html`: main page layout
- `src/styles/`: split styles for base, layout, and UI components
- `src/scripts/store.js`: localStorage persistence
- `src/scripts/utils.js`: note formatting, sorting, and filtering helpers
- `src/scripts/ui.js`: DOM rendering and form helpers
- `src/scripts/app.js`: app state and event wiring

## Usage

Open `index.html` in a browser to use the website.
